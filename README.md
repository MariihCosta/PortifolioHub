# PortfolioHUB

Repositório criado para o desafio de versionamento utilizando Git e GitHub.

## Objetivo
Desenvolver um portfólio acadêmico e profissional utilizando boas práticas de versionamento.

## Tecnologias Utilizadas
- HTML
- CSS
- Git
- GitHub
- GitHub Pages

├── projetos-pessoais/

## Funcionalidades
- Página web de apresentação
- Organização de projetos
- Versionamento com Git
- Publicação com GitHub Pages
